Simple Game

Introduction

Simple Game based on Phaser 3 and hosted on GitHub Pages.

Objective

Collect keys by killing enemies to exit the levels
You only need to kill one Enemy to exit a level.

Controls

Move using the WASD keys

You can Pick up rocks using the E key

Rocks can be put down for later use by pressing E, they can be stacked on eachother for more compact storage.

Throw rocks by pressing SPACE

Items

Syringe with stimulants, increases awareness.
Poisnous flower, slows you down.
Rock, brittle, can be picked up and thrown.
Boulder, very hard, can't be broken.
